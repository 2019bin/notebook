#中国城市大洗牌
本文主旨：未来房地产30年的分析判断
##区域经济格局
###超级城市
####上海
- 上海效仿日本，疏散非核心功能，成为综合性城市
- 上海同东京的性质差不多，处于竞争关系
####北京
- 北京现在定位为**全国政治中心，国际交往中心**
- 北京脱离经济功能，引入到雄安新区
- 北京将会有巨大的财政赤字
####珠三角地区
- 粤港澳大湾区的建立
- 打破政治阻碍，通过优化政策，加强三地联系

###二线城市
马太效应：强者愈强，弱者愈弱
大城市就是最后的赢家
####恶之花
- 房价飙升--沿海用地指标小，跟自身的经济体量不匹配，导致“挤牙膏”的卖房，对住宅供应有限，跟不上人口流动速度
- 户籍堡垒将低端劳动力进行阻碍，大部分农民工为大城市耗费一生，但拿不到应有的公共服务，最终回到十八线城市，拉低全社会的消费能力，加剧了阶层固化
####城市病
- 扩张大城市的征地指标
- 一二线城市扩张阻碍拆除，马太效应加剧
####中小城市
中小城市城市化下降，流水人口加剧。大城市吃下中小城市，成为了卫星城区

###中小城市
####问题
- 财政严重赤字
- 投资效果锐减
- 经济效益下降

###政绩考核
####地方造假
#####方式
- 空转：
一笔财政资金入库，建立一个虚假的项目，把钱转入这个项目，再把这笔钱拿出来，来来回回都是同一笔资金
- 虚增：
虚构应税事项和纳税依据
#####代价
- 影响中央对地方经济形式的判断和决策
- 中央对地方转移支付规模减小，收入变少
####造血能力
地方造血能力下滑
- 一般公共预算收入
原因：中国经济走势换挡中低档，地方财政收入增速碰上了缓速带，形成了大手大脚的毛病
- 土地出让收入
原因：
  - 房产调控
  - 中央定调房住不炒，大力发展租凭市场
  - 去杠杠的背景下，中央收缩流动性，过往的投资驱动方式难以继承
- 地方债务
中央制止地方政府违法违规举债，地方融资要么缩紧，要么受限

###南北地区差异
####重点城市的趋势
- 东西部之间的差距，正在缩小
- 南北部之间的差距，正在拉大
####人才与资金流动
- 本外币存款余额（反映是一座城市对资金的吸附能力）
南方比北方城市多，且增速比北方快
- 人口净增
南方城市吸引了大量人口
- 人口老龄化
北方城市老龄化程度比南方严重得多
####原因
- 地理因素
   - 温带地区有舒适的温度，适度的降水，适合人类的经济活动
   - 沿海的气候条件比内陆更为优越
- 中国经济转型
   - 计划经济初期，北方重工业撑起中国经济，国有经济比较大，转型困难
   - 改革开放，南方城市跟全球经济关联，经济发展比北方快

###楼市分化
####小城市全面日本化
- 结构性危机
- 老龄化，少子化
- 出现空屋，形成鬼城
####大城市全面香港化
- 房价过高，超出当地居民的承受能力
- 富豪抢购楼盘，坐等投机
####解决
中国用经济增长拉高居民收入，以此慢慢消化掉泡沫
###抢人大战
####创业
- 最好找的东西就是钱，最难找的是让钱生钱的方式
北京，深圳，上海，杭州，南京，广州的人均存款余额处于前列
- A股上市公司数量（反映一个城市的市场化程度）
上市公司越多越不慌，越靠后的城市越饥渴，越对本科毕业生条件越宽松
  - 上市公司处于中下游的地方，大中专生就能无门槛落户
  - 一线城市非985.211不接受
  - 超一线城市只要长江学者，院士，万人计划
####精致主义者
- 人均本外币存款：厦门，珠海
- 人文主义色彩：杭州，南京
####公务员
- 守成型人格：
   - 教育医疗为主的（注重里子）：广州，杭州
   - 基建为主的（注重表字）：深圳，珠海，郑州
- 建设者人格
祖国最需要的地方：广阔落后地区

##北京，天津，河北
###北京的定位
####赶人运动
- 整改“拆墙打洞”--清理外来人口
- 建设通州城市副中心--疏散公务员
- 规划雄安新区--迁移各类事业单位的人
后果：gdp与地方税收的流失，但北京非首都的疏散还是进行，因为北京定义为中央的北京
####北京呈圆心
越靠近圆心资源就集聚
北京具有最好的教育机构，最好的民生医院，最有权势的衙门，最有地位的国企
整个城市如同一个同心圆，越靠近圆心的位置，资源就越多
####定位
四个定位：
- 全国政治中心
对内功能，服务于中央党政军三套班子，保障国家政务活动安全高效运行
- 文化中心
承接过去，在一个蕴含古老中华文明城市，孕育出一股指引世界文明方向的科技力量
- 国际交往中心
对外服务，服务于重大外交外事活动
- 科技创新中心

###北京财政问题
####问题：
- 北京做减量运动，疏散非首都功能
- 房地产投资重要性降低
- 大幅度减税
####财政支出：
- 中央宣布大减税政策，财政收入变少
- 未来北京举报多次大型活动

###天津抢人
####天津的问题：
- 人口负增长
- 税收收入负增长
- 企业的存款增量逐年降低

####投资驱动问题：
- 经济活动跟不上投资驱动时，会出现“鬼城”这一城市现象
- 无法获得长期的国家资源倾斜

####政府做法：
- 高考福利
- 压低房价
- 下降非税收入，积极调整财政结构

###雄安新区
####特色
- 租房为主的城市
  - 政府提供租凭租房，保障入城就有房屋保障
  - 租购同权，买房需要在这区域工作时间和其他方面的条件
  - 楼市半冻结，房子必须持有20年以上才能出售，并在出售时政府具有优先购买权

- 公民社会
  - 可塑造性空间大（城市人口可容纳250万人，体量小）
  - 人口素质高（具有高学历高收入的青壮年）
  - 自主性强（继浦东，滨海之后又一个具有全国意义的城市，地方立法性充足）

- 智慧城市
  - 运用高科技技术改变生活（大数据，ai，无人自动化）
  - 打破科学发展与伦理问题的局限

###燕郊
####燕郊房价回涨原因：
- 北京城市副中心正式启用
- 连接燕郊和北京中心城区的地铁“平谷线”正式敲定

####京津冀协同发展
- 解决燕郊的被房地产绑架的问题
- 消除环京贫困带，带领河北共同富裕

##华南地区
###广深港高铁
####铁路重要性
- 使心理上的距离大幅缩小（香港深圳半小时生活圈，香港广州1小时生活圈）
- 一个强化国家存在的工具

####影响
- 拉高周边的地价
- 粤港澳的经济整合
- 拉高技术创新

####意义
- 粤港澳大湾区在“智造”方面上带动全球产业链，促进产业升级
- 全球分工体系重构，粤港澳湾区地位上升
- 在中美贸易争端的大背景下，粤港澳大湾区有可能成为中国打破西方高技术围堵的一个突破点

###港珠澳大桥
####意义
- 打破高端技术的垄断，掌握定价权和话语权
- 有利于港澳长期繁荣稳定

####国际科技创新中心
- 关税不合理阻碍创新设备流通
- 资金流动性较差
- 相关人员通关手续麻烦

####差距
- 环境问题
- 科技创新整体实力与产业升级

####特权
- 互联网特区
- 湾区人率先成为“合格境内个人投资者”
- 湾区人享有更大的自主权

###深圳问题
- 企业利润低，甚至半年利润买不起一套房
- 披着工业区改造的皮，干房地产的生意
- 工业用地紧张，但办公用地租金飞升

###汕头失落
####过往的优势
- 政策优势（1980年汕头是经济特区）
- 贸易优势（临近西大平洋国际黄金航道）
- 侨乡优势（潮汕在海外有1000多万华侨）

####问题
- 环境问题（河流污染及其严重）
- 逃税漏税非常严重
- 造假严重
- 赌博事业发达

###海南——自贸港
####重心不在金融和博彩中心
- 中央指出，海南“不以转口贸易和加工制造为重点”
- 内地反腐，博彩业有下行势头
- 赌场，跑马这种博彩业很复杂，带来大量问题
- -起点很低，没当地资源很难运转

####试验田
- 相对独立的地理位置
- 经济规模较小，但是一个完整的地理单位，产生负面影响较小
- 人口充足

##长三角地区
###进口博览会
####背景
- 国内经济显现疲态（债务危机，产能过剩...突出矛盾逐渐出来，不确定性上升）
- 全球经济增速放缓，中美摩擦程度越来越严重,新“冷”战的味道越来越浓
- 中央对外部环境的研判从“明显变化”到“深刻变化”，非常务实

####释放信号
- 中国对“开放”是主动积极的，是内部经济发展需求，不是谁逼的
- 中国的开放还在进一步扩大，没有止步不前
- 最高领导人亲自表态，中国对开放的重视程度不容置疑

####中国树立进口大国形象重要性
- 中国必须坐稳世界第一贸易大国的位置，才能占据国际话语权，才有影响国际贸易规则
- 中国必须要转型升级

####选择上海的原因
- 上海靠近长江，长江直达中国最广阔的疆域，具有庞大的消费市场
- 上海要打造一个具有全球竞争力的亚太中转枢纽
- 加快长江三角洲区域一体化发展

###g20在杭州
- 互联网融入生活当中，“只带一部手机生活得很好”
- 运用流量入口意识（用免费的方式获取流量，有流量就有其他收益）
- 重点发展第三产业，引入阿里巴巴这类的互联网大品牌
- 具有快节奏的高科技收益的区域
- 普惠金融（城市生活的便捷性与信用和支付有关）

###"佛系"杭州
- 杭州靠土地财政撑腰
  原因：
    - 基础设施投资力度大
    - 2022年亚运会

###苏州外资撤退
####苏州特点：
- 苏州模式：以新加坡工业园区作为代表，依靠外商合资，合作，独资等方式带动经济增长，走上出口导向型道路。
- 依靠人口“红利”，工业园区特殊的管理模式，优惠的税收政策获得技术外溢
- 外资制造业是苏州的主要部分
####风向标
- 富士康南下印度
- 松下公司（首家进入中国的外企）在撤离
- 官媒喊话别让外资跑了
####问题
- 人工成本升高
- 园区房价升高，企业用地成本加剧
- 物价受到侧供给改革，原材料价格飞涨
- 高税率
- 在“中国制造”到“中国智造”方向迷失

###亩产论英雄
**内容**：对亩产收益高的企业，要什么给什么，排污权指标多给，用电用水优惠价。。。。
**原因**：过去三十年间，中国长期追求“高速度”而忽视“高质量”

###浙江政务改革
传统信息传达恶果：
- 产生信息延迟
- 信息失真
####自我革命
- “浙政钉”app让省，市，县，乡，村五级机构在上边实现在线联动
- 通过政务平台降低办理手续步骤

###杭州大湾区
####出现条件
- 接近实现“数字经济”
- 软硬件基础：融合支付，借贷和投资的强大app，一种全新的金融服务形式
- 人工智能，区块链专利创新能力强

##抢房潮
###房地产调控
国内不少地方严格限制一手新房的销售价格，导致一手房政府核定销售价大大低于同期二手房市场成交价

###二线城市出现房荒
- 房地产开发商拖延入市，捂盘惜售
- 开发商拉长施工周期，延缓项目竣工
- 政府出了“限价”的烂招
- 规划大量非住宅区域，严重挤压住宅

###房子质量下降
- 入手低质楼：建材不合格，工期不合格
- 接盘烂尾楼：高周转
- 房价下降

###租客的末日
####房租上涨
- 资本圈地（资本拉杠杠和设定定价权）
- 群租房大整改（拆除违法建设）
- 租金回报率低
###中国式棚改
- 通过了PSL（抵押补充贷款），棚改具有一定资金

###改变“限购”
####原因
- 户籍制度严重阻碍大城市发挥城市效应
- 经济下行压力下，地方政府大规模减税，财政收支紧张，导入人口，利用房地产

####路劲：
- 在Ⅰ型大城市中，文凭就是购买房子的基础
- 利好广深郊区楼盘
